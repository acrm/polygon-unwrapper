Пример вызова утилиты из командной строки:
	PolygonUnwrapper.exe input.obj 1:100
где 
	«input.obj» — путь к исходной модели в формате obj, обязательный параметр
	«1:100» — диапазон загружаемых полигонов, здесь с 1-го по 100-й. Если указывать вместо диапазона просто число,
	то оно будет считаться верхней границей. Необязательнвый параметр, без него будут выгружены все полигоны.

В результате в папке с утилитой создаются два файла:
	«model.obj» — исходная модель, в которой каждый полигон заключён в нумерованую группу.
	«grid.obj» — те же полигоны в тех же нумерованных группах, но расположеных плоско на квадратной сетке.
